VXI - Privacy Policy

1. Information We Collect

VXI may collect limited data required for operation, including:

  • Server ID
  • Channel ID
  • User ID (only when necessary for commands or moderation)
  • Message content (only to process commands, never stored long-term)
  • Server configuration data (prefixes, settings, roles, etc.)

VXI does NOT collect:

  • Email addresses
  • Passwords
  • Direct Messages (unless you actively use bot commands in DMs)
  • Personal identifiable information outside Discord

2. How Data Is Used

Collected data is used to provide bot functionality, including:

  • Executing commands
  • Managing server settings
  • Moderation actions
  • Preventing abuse and spam

Data is never sold or shared with third parties.

3. Data Storage

Data (such as configuration) may be stored in a secure database. Data is kept only as long as required for bot functionality.

You may request deletion of your server’s data at any time.

4. Data Sharing

We do not share or sell any data. Only the bot owner/developers may access internal logs if needed.

5. Children’s Privacy

VXI does not knowingly collect data from users under 13. If you are under 13, do not use the Bot.

6. Data Removal Requests

If you want your server data deleted, contact the bot owner on Discord.

7. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of VXI means you accept the updated policy.

8. Contact

You may contact the bot owner via Discord for questions or concerns.
